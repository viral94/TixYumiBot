Данный код используется для работвы чаи бота на Tixchat.com. Для начала работы с ним нужно просто запустить консоль в браузере с открытым чатом и загрузить в неё скрипт. Данный бот был разработан Юзером с ником Vaflan. Ему за это отдельная благодарность. Мною бот был сильно изменён, добавлен функционал. Бот может теперь отвечать на комментарии приветствовать, прощатся и вести простой диалог. Кидать раз в час рандомные комментарии(данная функция используется для рекламы приложения и канала на YouTube). Также бот может порекомендовать аниме по жанру. Мною также добавленна антимат система. Она удаляет матерные комментарии в чате(нужно наделить бота правами редактировать текст в чате). Антимат система требует доработки. Бот не распознаёт матерные слова в контексте.
